## 用自己的话回答

- 什么是特征，什么是标签：特征是模型用来做预测的输入信息（如用户使用月数、投诉次数等），标签是模型要预测的目标答案（如用户是否流失）。类比来说，特征是考试题目，标签是标准答案，模型通过大量题目-答案对来学习解题规律。
- 为什么要保留测试集：测试集模拟模型从未见过的新数据。如果把所有数据都用于训练，模型可能死记硬背训练数据（过拟合），在新数据上表现很差。保留一部分数据不参与训练，用来最终评估模型的真实泛化能力，就像考试不能考已经做过的原题一样。
- 为什么83%准确率仍可能没有用：如果数据集本身就不均衡（比如只有17%的用户流失），模型可以偷懒永远预测"不流失"，这样准确率确实是83%，但实际上一个流失用户都没识别出来。这种"高准确率"是虚假的——模型没学到任何有价值的东西，对业务毫无帮助。
